Worksheet 11 - Motor PWM

1. Introduction

Pulse Width Modulation (PWM) is a method used by PLCs to control motor speed by rapidly switching an output on and off. The ratio of on-time to off-time determines how much power the motor receives. This worksheet explores how PWM is used to vary the speed of a DC motor.

2. Over To You

Follow these steps to explore PWM motor control:

1

Open Worksheet 11

Set the potentiometer to 0% (fully anticlockwise). Press Start.

2

Increase Speed

Rotate the potentiometer clockwise and observe motor speed increase. Watch the PWM Duty Cycle on the HMI.

3

Full Range Test

Move from 100% to 0% a few times and observe the trend of the scaled input and the duty cycle.

3. So What?

Why this matters

You’ll see PWM used everywhere to vary motor speed without analogue output cards. Duty cycle directly affects motor power—100% is fully on, 0% is off. Understanding the link between an analogue input (potentiometer) and the PWM output builds core diagnostic skills.

Duty cycle in practice

As you turn the potentiometer up, the PWM duty cycle increases and the motor speeds up. If the motor doesn’t respond, check the output wiring and the HMI trend.

Diagnostic tip

Compare the scaled input %, the PWM % on the HMI, and the motor speed. If input and PWM change but speed doesn’t, suspect the motor or the power stage.

Key Takeaways

  • PWM controls speed by switching outputs quickly.
  • Duty cycle (%) sets power delivered to the motor.
  • The potentiometer sets PWM level via a 0–10 V input.
  • Reading HMI trends helps diagnose speed issues.
